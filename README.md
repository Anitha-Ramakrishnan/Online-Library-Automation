# Online Library System (Demo)

Simple demo website with login, hashed passwords, persistent JSON-backed storage via lowdb, forgot-password (demo reset), library, cart, and checkout.

Quick start:

```powershell
cd "C:\Users\lvign\OneDrive\Documents\Projects\GITHub_Projects\Online_LibrarySystem"
npm install
npm start
```

Then open http://localhost:3000

Demo credential:
- username: user1
- password: password1

Forgot password will reset the user's password to `reset123` (demo only).

Data is stored in `data/library.json`.
